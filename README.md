## Prerequisites

- Node.js (v14 or higher)
- Composer

## Getting Started

## Clone the repository:

git clone <repository_url>
cd project-folder

## Install the dependencies:
composer install
npm install

## Create a .env file
cp .env.example .env

## Generate an application key:
php artisan key:generate
php artisan serve
## Development
npm run dev

Access your application at http://localhost:8000

## Build
npm run build
